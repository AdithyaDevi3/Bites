# Bites Dependency Requirements

## Frontend (Node)

Install from the repository root:

```bash
npm install
```

Run the frontend:

```bash
npm run dev
```

## Backend (Spring Boot)

Install and run from the `Spring/` directory:

```bash
cd Spring
mvn spring-boot:run
```

## Notes

- Frontend dependencies are declared in `package.json`.
- Backend dependencies are declared in `Spring/pom.xml`.
